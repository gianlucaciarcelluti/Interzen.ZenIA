## Avvio Rapido (5 minuti)

### Che cos'è MS14-AUDIT?
MS14-AUDIT fornisce funzionalità di audit e tracciamento per la piattaforma ZenIA.

### Responsabilità principali
- Funzionalità core secondo le specifiche architetturali
- Integrazione con gli altri microservizi
- Raccolta e gestione dei log di audit
- Supporto per monitoraggio e reporting di conformità

### Primi passi
1. Consulta [SPECIFICATION.md](SPECIFICATION.md) per le specifiche tecniche dettagliate
2. Controlla `docker-compose.yml` per il setup locale
3. Rivedi [API.md](API.md) per gli endpoint di integrazione
4. Esplora la cartella [examples/](examples/) per esempi di request/response

### Stack tecnologico
- **Linguaggio**: Python 3.10+
- **API**: FastAPI
- **Database**: PostgreSQL
- **Cache**: Redis
