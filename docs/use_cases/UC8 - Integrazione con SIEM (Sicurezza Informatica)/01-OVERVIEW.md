# UC8 - Sottoprogetti per Integrazione con SIEM

## Overview

Integrazione con SIEM: raccolta log sicurezza, elaborazione eventi, archiviazione e analitiche security intelligence.

## Sottoprogetti (SP)

### SP38 - Collettore SIEM
- Raccolta log da sistemi eterogenei
- Connessione a sorgenti dati security
- Normalizzazione formati log

### SP39 - Elaboratore SIEM
- Elaborazione eventi security
- Correlazione anomalie
- Rilevamento minacce

### SP40 - Archiviazione SIEM
- Archiviazione eventi security
- Gestione retention log
- Query performance su archivi grandi

### SP41 - Analitiche SIEM e Reporting
- Analitiche security intelligence
- Dashboard minacce
- Reporting compliance security
