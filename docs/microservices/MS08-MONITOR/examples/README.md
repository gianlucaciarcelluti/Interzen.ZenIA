# Esempi JSON - MS08-MONITOR

Questa directory contiene esempi JSON per le principali strutture dati utilizzate da MS08-MONITOR.

## File Inclusi

- `metrics-query-response.json`: Risposta query metriche Prometheus
- `log-entry-structured.json`: Entry log strutturato
- `trace-span-example.json`: Esempio span trace Jaeger
- `alert-firing-example.json`: Alert in stato firing
- `dashboard-config.json`: Configurazione dashboard Grafana
- `api-health-response.json`: Risposta health check API
- `bulk-metrics-upload.json`: Upload bulk metriche
- `alert-rule-definition.json`: Definizione regola alert
- `notification-webhook.json`: Payload notifica webhook
- `audit-log-entry.json`: Entry log audit
